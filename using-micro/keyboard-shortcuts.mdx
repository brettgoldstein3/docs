# General Shortcuts

| **Action** | **Shortcut** |
| --- | --- |
| Create new view | Shift + E |
| Navigate between views (right) | Option + Tab |
| Navigate between views (left) | Option + Shift + Tab |
| Navigate between apps | Option + 1 - 9 |
| Show/hide filters | ⌘ + Shift + ↓ |
| Open command car | ⌘ + K |
| Open search | \ |
| Create new record | E |
| Create new task | T |
| Toggle right sidebar open/closed | ⌘ + / |
| Toggle left sidebar open/closed | ⌘ + . |
| Copy page link | ⌘ + L |
| Go to primary action | Enter |

# Grid Shortcuts

| **Action** | **Shortcut** |
| --- | --- |
| Navigate to the cell to the left | ← |
| Navigate to the cell to the right | → |
| Navigate to the cell or row above | ↑ |
| Navigate to the cell or row below | ↓ |


<Note>Shortcuts use the ⌘ key on Mac. On Windows/Linux, use Ctrl instead of ⌘.</Note>
