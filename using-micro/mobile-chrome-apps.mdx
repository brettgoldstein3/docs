# Chrome Extension

The [Micro Chrome Extension](https://chromewebstore.google.com/detail/microso-chrome-extension/ejibgfefhiaomijbkpcbeppeblaemfdm) enables seamless LinkedIn integration with Micro.

## What It Does

The Chrome extension automatically handles LinkedIn authentication, allowing you to:

- **Import LinkedIn connections** - Sync your LinkedIn network to Micro
- **Enable LinkedIn messaging** - Send and receive LinkedIn messages within Micro
- **Update activity automatically** - Keep your LinkedIn data synchronized

## Installation

<Steps>
<Step title="Install from Chrome Web Store">
  a. Visit the [Micro Chrome Extension](https://chromewebstore.google.com/detail/microso-chrome-extension/ejibgfefhiaomijbkpcbeppeblaemfdm) page
  
  b. Click **"Add to Chrome"**
  
  c. Confirm the installation when prompted
</Step>

<Step title="Connect to Micro">
  a. Click the puzzle piece icon in your Chrome toolbar
  
  b. Find and click on the **"Micro"** extension
  
  c. Click **"Connect"** to authenticate your LinkedIn account
  
  d. Follow the authentication prompts
</Step>
</Steps>


## Troubleshooting

### Extension Not Installing
- **Check Chrome version** - Ensure you're using Chrome 88 or later
- **Clear browser cache** - Try clearing your browser cache and cookies
- **Disable other extensions** - Temporarily disable conflicting extensions
- **Try incognito mode** - Test installation in Chrome's incognito mode

### LinkedIn Authentication Issues
- **Check LinkedIn login** - Ensure you're logged into LinkedIn in Chrome
- **Clear LinkedIn cookies** - Clear LinkedIn-specific cookies and try again
- **Reinstall extension** - Remove and reinstall the extension
- **Contact support** - Reach out to [support@micro.so](mailto:support@micro.so) for help

### Connection Sync Problems
- **Refresh Micro** - Reload the Micro application
- **Check permissions** - Verify the extension has necessary permissions
- **Manual sync** - Try manually triggering a sync from Micro
- **Update extension** - Ensure you have the latest version

### Messaging Not Working
- **Verify connection** - Check that LinkedIn is properly connected
- **Refresh page** - Reload the Micro interface
- **Check LinkedIn status** - Ensure LinkedIn messaging is available
- **Clear cache** - Clear browser cache and try again

### Performance Issues
- **Disable other extensions** - Other extensions may conflict
- **Check memory usage** - Close unnecessary browser tabs
- **Update Chrome** - Ensure you're using the latest Chrome version
- **Restart browser** - Sometimes a simple restart fixes issues
